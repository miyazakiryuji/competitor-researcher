# 環境チェック（doctor）

あなたは `competitor-researcher` プラグインの環境チェック担当です。以下を順番に、日本語で実行
してください。**何も変更しない**（読み取りとチェックのみ）。

## 1. スクリプトによるチェック

Bash ツールで次を実行し、出力をそのままユーザーに見せる:

```bash
python3 "${CLAUDE_PLUGIN_ROOT}/scripts/doctor.py" [$ARGUMENTS があればそのフォルダ]
```

（Python・PDF化用ブラウザ・作業フォルダ・自社サービス.md の記入状況を確認する）

## 2. Web検索のチェック

WebSearch ツールで軽い検索を1回だけ実行して（例: 「Claude Code」）、結果が返るか確認する。

- 返ってきた → 「✅ Web検索 — OK」
- ツールが使えない／拒否された → 「⚠️ Web検索が使えません。materials/ に手元資料を
  置けば調査できます（レポートには『手元資料のみで作成』と明記されます）」

検索結果の中身は表示しない（可否の確認だけ）。

## 3. まとめ

最後に3行以内でまとめる:

1. 総合判定（例: 「準備OKです！」／「⚠️が2件ありますが、当日サポートで解決できるレベルです」）
2. ⚠️ があれば、直し方を1行ずつ
3. 次の一歩（未初期化なら `/competitor-researcher:init`、初期化済みなら `/competitor-researcher:research`）
