# はじめに（専属リサーチャーを使う3ステップ）

これは「**競合の料金と特徴を Web で調査して、比較レポートまで自動で作る**」
Claude Code プラグイン（competitor-researcher）です。
**GitHub のアカウントは不要**です。次のどれか1つで取り込めます。

## 取り込み（どれか1つでOK）

### A. GitHub から（おすすめ・コピペ2行）
Claude Code の中で:
```
/plugin marketplace add miyazakiryuji/competitor-researcher
/plugin install competitor-researcher@competitor-researcher
```
※ 一覧に出てこない場合だけ `/plugin enable competitor-researcher@competitor-researcher`

### B. 配布フォルダから（恒久インストール）
Claude Code の中で:
```
/plugin marketplace add "<このフォルダのパス>"
/plugin install competitor-researcher@competitor-researcher
```
※ zip で受け取った場合は、**先に解凍**してから、できたフォルダのパスを指定してください。

### C. いちばん手軽（そのセッションだけ有効）
ターミナルで、このフォルダ（zip のままでもOK）を指定して起動するだけ:
```bash
claude --plugin-dir "<このフォルダ か zip のパス>"
```

---

## 使う（3ステップ）

1. `/competitor-researcher:init`
   … 作業フォルダ（自社サービス.md・リサーチの好み.md・reports/ など）を作ります。
2. `自社サービス.md` を記入
   … 分かるところだけでOK。すぐ試すなら `サンプル_自社サービス.md` をコピー。
3. `/competitor-researcher:research`
   … 競合候補が提案されるので番号で選ぶと、`reports/` に
   出典付きの比較レポート（.md）、グラフ入りのHTMLダッシュボード（.html・
   ダブルクリックでブラウザに開ける1枚もの）、そのPDF資料（.pdf）ができます。

## 前提

- Claude Code（最新版・有料プラン）
- 料金調査に Web 検索を使います（使えない環境では `materials/` に手元資料を置けば動きます）

## 2つだけ注意

- レポートの料金には**出典と確認日**が付きますが、大事な意思決定の前には出典URLをご自身でも確認してください。
- 実在のお客様の名前・機密情報は、作業フォルダに置かないでください。
